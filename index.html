Tentang Program

Apa itu
Katalis?

Katalis — akronim dari Kaderisasi Tahap Awal Duta Kemahasiswaan — adalah program rekrutmen resmi Badan Eksekutif Mahasiswa (BEM) Fakultas Hukum Universitas Indonesia untuk menjaring fungsionaris terpilih yang akan menggerakkan organisasi selama satu periode kepengurusan.

"Menjadi bagian dari Katalis bukan sekadar jabatan, melainkan sebuah komitmen untuk menjadi agen perubahan di lingkungan Fakultas Hukum Universitas Indonesia."— Project Officer Katalis BEM FH UI 2026
Visi Katalis
Membentuk fungsionaris yang adaptif, kritis, dan berintegritas.
Nilai Hukum
Berlandaskan nilai akademik dan etika kemahasiswaan FH UI.
Kolaborasi
Membangun sinergi antar mahasiswa lintas angkatan.
BEM Fakultas Hukum UI
Organisasi kemahasiswaan tertinggi di Fakultas Hukum UI yang menjalankan fungsi advokasi dan representasi mahasiswa.
